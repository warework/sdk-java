What can I find in your servers?
Warework provides different types of repositories where to download or use software libraries and its associated resources:
  • User guides repository: Every software developer guide published by Warework is in the form of a PDF file. Check out how to access each PDF guide here.

  • On-line tutorials repository: Software developer guides can also be reviewed as HTML pages. The content is the same as the User guides but the repository is a different one. Check out how to access each on-line tutorial here.

  • API repository: Technical information about the programming interface of each library is avaliable in HTML format. Review how to access each API here.

  • Change logs repository: Historical changes made to a software library are registered in plain text files. Check out how to access each log file here.

  • Maven 2 repository: Warework provides a Maven 2 repository where Java developers can link their projects to every Warework library created for the Java programming language. For further information about how to access the Warework Maven Repository, please refer to this FAQ.

  • JavaScript repository: Web developers can link to every Warework JavaScript library in this repository. You can find out the required HTML code to import each one in the description page of a library or the README file associated to it.

  • XSD repository: Warework XML files that require schema definition and validation should point to http://repository.warework.com/xsd. Follow the instructions in the user guide of a software library that uses a specific XML Schema to check out the full URL required.
